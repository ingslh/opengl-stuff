1. 使用jsx导出ae的shape后，会包含用于生成贝塞尔曲线的数据，每个顶点包含三个数据（切入向量、切出向量、顶点位置），形式如下：
```json
"Vertices": [
      [
        -8.21299743652344,
        53.7674560546875
      ],
]
```
2. 图形对应关系为ae中每一个shape对应的group（group内包含path、fill属性），因此每个shape可能对应多个渲染的shader，即每一个group（准确的说是path）需要有一个shader对应。
